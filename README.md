🍔 Zomato & Swiggy Order Analyzer
A Python-based data engineering and analytics project that generates a highly realistic synthetic dataset of food delivery transactions and utilizes Pandas to extract deep behavioral insights.

📝 Project Overview
This project was built to demonstrate proficiency in Data Generation, Feature Engineering, and Advanced Data Aggregation.
Instead of relying on a static, pre-cleaned CSV, this script dynamically generates 200 realistic transaction logs (using NumPy's random distribution tools) complete with varying bill amounts, delivery charges, discounts, and date-time stamps. It then processes this raw "Bronze" data through a pipeline to uncover actionable spending habits.

🛠️ Key Technologies & Skills Demonstrated
Python & NumPy: Designed a robust synthetic data generator that maps specific cuisines to real-world Indian restaurant chains and introduces realistic probability distributions for discounts and delivery fees.

Pandas Data Manipulation:
Feature Engineering: Converted raw date strings into Pandas Datetime objects to dynamically extract Month, Day_Of_Week, and a custom Is_Weekend boolean column.

Vectorized Math: Calculated net out-of-pocket costs without using slow for loops.
Advanced Aggregations: Utilized .groupby() and .agg() to compute multiple metrics (Count, Sum, Mean) simultaneously across various categories.

📊 Key Insights Generated by the Pipeline
The script outputs a comprehensive terminal report that answers the following business/personal questions:
Platform Battle: Which platform (Zomato vs. Swiggy) captures the majority of the lifetime delivery spend?
Top Frequented Restaurants: Ranks the top 3 favorite restaurants by order volume and total money spent.

Cuisine Breakdown: Analyzes which food category (North Indian, Chinese, Fast Food, etc.) has the highest Average Order Value (AOV).
Ordering Patterns: Compares average bill size and order frequency between Weekdays and Weekends.
Payment Efficiency: Analyzes if certain payment methods (UPI, Credit Card, Net Banking) correlate with higher discounts or lower delivery fees.
